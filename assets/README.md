# Assets

Diagrams for the project. The pipeline and architecture diagrams are currently
maintained as ASCII art inline in [`../README.md`](../README.md) and the
[`../docs/`](../docs/) files so they stay version-controlled and reviewable in
plain text.

Rendered image versions (`compiler_pipeline.png`, `architecture.png`) can be
added here and embedded in the README when a graphical format is preferred.
